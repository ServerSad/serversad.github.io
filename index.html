ServerUi Documentation

Biblioteca de interface para criação de GUIs no Roblox

Introdução

Esta documentação ensina como utilizar todas as funcionalidades da biblioteca ServerUi para criar GUIs completas e organizadas.

Suporte a ícones LucideBlox: Clique aqui para ver os ícones

Importando a biblioteca

local ServerUi = loadstring(game:HttpGet("https://raw.githubusercontent.com/ServerSad/UiLib/refs/heads/main/Lib/uilib.lua"))()

Sistema de Cargos (Roles)

ServerUi:MakeRoles({
    Especial = {
        Color = "#FFFF00",
        Users = {11223344, "SeuNome", "NomeDoMeuAmigo"}
    },
    User = {
        Color = "#00FF00",
        Users = "everyone"
    }
})

Explicações:

  • NomeDoCargo: Identificador do cargo (ex: "Especial")
  • Color: Cor hexadecimal aplicada no nome do usuário
  • Users: Lista de IDs ou nomes de usuários que pertencem ao cargo
  • "everyone": Aplica o cargo a qualquer jogador que usar o script

Criando a Janela Principal

local Window = ServerUi:MakeWindow({
    Name = "Minha GUI",
    SaveConfig = true,
    ConfigFolder = "MinhaPasta",
    IntroText = "Bem-vindo!",
    IntroEnabled = true,
    ShowIcon = true,
    Icon = "rbxassetid://12345678",
    IntroIcon = "rbxassetid://12345678",
    SearchBar = {
        Default = "🔍 Procurar...",
        ClearTextOnFocus = true
    },
    KeySystem = true,
    KeySettings = {
        Title = "🔐 Key System",
        Subtitle = "Digite sua chave para continuar",
        LoadingTitle = "Carregando",
        LoadingSubtitle = "...",
        Author = "ServerUI",
        Textbox = "Insira aqui a key...",
        Button = "Confirmar",
        KeyButton = "SeuLink",
        Note = "Pegue a key clicando no icone de chave na direita!",
        FileName = "MinhaKey.txt",
        SaveKey = false,
        Key = {"KeyTeste"},
        Getfromsite = "https://seusite.com/minhaskeys.txt",
        UseServerKey = true,
        ServerKey = "meu-token",
        ServerKeyPadrao = "Key-123"
    }
})

Explicações:

  • SaveConfig: Habilita salvamento automático
  • ConfigFolder: Pasta onde as configurações serão guardadas
  • IntroEnabled: Mostra uma introdução animada
  • SearchBar: Ativa campo de busca para abas
  • Icon / IntroIcon: Ícones da GUI

Criando uma Aba (Tab)

local Tab = Window:MakeTab({
    Name = "Minha Aba",
    Icon = "rbxassetid://4483345998",
    RequiredRole = "AllRoles"
})

Controle de Acesso com Cargos:

  • RequiredRole: Define quem pode ver a aba
  • Pode ser:
    • Uma string: "Admin", "User", etc.
    • Uma lista: {"Admin", "Mod"}
    • Ou "AllRoles" para permitir acesso a todos

Botão

Tab:AddButton({
    Name = "Clique aqui",
    Callback = function()
        print("Botão clicado!")
    end
})

Toggle (Interruptor)

Tab:AddToggle({
    Name = "Ativar algo",
    Default = false,
    Flag = "meuToggle",
    Save = true,
    Callback = function(state)
        print("Toggle:", state)
    end
})

Slider (Deslizante)

Tab:AddSlider({
    Name = "Volume",
    Min = 0,
    Max = 100,
    Default = 50,
    Increment = 5,
    ValueName = "%",
    Flag = "sliderVolume",
    Save = true,
    Callback = function(value)
        print("Volume:", value)
    end
})

Dropdown (Lista suspensa)

Tab:AddDropdown({
    Name = "Escolha uma cor",
    Options = {"Vermelho", "Verde", "Azul"},
    Default = "Vermelho",
    Flag = "corSelecionada",
    Save = true,
    Callback = function(option)
        print("Cor escolhida:", option)
    end
})

Bind (Tecla de Atalho)

Tab:AddBind({
    Name = "Tecla rápida",
    Default = Enum.KeyCode.F,
    Hold = false,
    Flag = "teclaRapida",
    Save = true,
    Callback = function()
        print("Tecla pressionada!")
    end
})

Hold = true: Executa somente enquanto a tecla estiver pressionada

Textbox (Campo de texto)

Tab:AddTextbox({
    Name = "Digite algo",
    Default = "",
    TextDisappear = true,
    Callback = function(text)
        print("Você digitou:", text)
    end
})

Label (Texto fixo)

Tab:AddLabel("Este é um texto fixo na interface")

Parágrafo (Texto longo)

Tab:AddParagraph("Título", "Texto explicativo.\nVocê pode usar quebras de linha.")

Log (Mensagem em destaque)

Tab:AddLog("Sistema iniciado com sucesso.")

Notificação (Popup)

ServerUi:MakeNotification({
    Name = "Título",
    Content = "Mensagem de conteúdo",
    Time = 3
})

Salvando Configurações

Se você definiu SaveConfig = true, basta rodar:

ServerUi:Init()

As configurações serão carregadas automaticamente nos próximos usos.

Usando Flags (Acesso a valores)

Leitura:

local valor = ServerUi.Flags["meuToggle"].Value

Alterar manualmente:

ServerUi.Flags["sliderVolume"]:Set(75)

Exemplo Completo

Implementação completa com todos os componentes:

-- Inicialização da biblioteca
local ServerUi = loadstring(game:HttpGet("https://raw.githubusercontent.com/ServerSad/UiLib/refs/heads/main/Lib/uilib.lua"))()

-- Configuração de cargos
ServerUi:MakeRoles({
    Especial = {
        Color = "#FFFF00",
        Users = {11223344, "SeuNome", "NomeDoMeuAmigo"}
    },
    User = {
        Color = "#00FF00",
        Users = "everyone"
    }
})

-- Criação da janela principal
local Window = ServerUi:MakeWindow({
    Name = "Minha GUI",
    SaveConfig = true,
    ConfigFolder = "MinhaPasta",
    IntroText = "Bem-vindo!",
    IntroEnabled = true,
    ShowIcon = true,
    Icon = "rbxassetid://12345678",
    IntroIcon = "rbxassetid://12345678",
    SearchBar = {
        Default = "🔍 Procurar...",
        ClearTextOnFocus = true
    },
    KeySystem = true,
    KeySettings = {
        Title = "🔐 Key System",
        Subtitle = "Digite sua chave para continuar",
        LoadingTitle = "Carregando",
        LoadingSubtitle = "...",
        Author = "ServerUI",
        Textbox = "Insira aqui a key...",
        Button = "Confirmar",
        KeyButton = "SeuLink",
        Note = "Pegue a key clicando no icone de chave na direita!",
        FileName = "MinhaKey.txt",
        SaveKey = false,
        Key = {"KeyTeste"},
        Getfromsite = "https://seusite.com/minhaskeys.txt",
        UseServerKey = true,
        ServerKey = "meu-token",
        ServerKeyPadrao = "Key-123"
    }
})

-- Aba principal (acessível a todos)
local Tab = Window:MakeTab({
    Name = "Principal",
    Icon = "rbxassetid://4483345998",
    RequiredRole = "AllRoles"
})

-- Aba especial (só para usuários com cargo "Especial")
local Tab2 = Window:MakeTab({
    Name = "Tab Especial",
    Icon = "rbxassetid://4483345998",
    RequiredRole = "Especial"
})

-- Notificação de boas-vindas
ServerUi:MakeNotification({
    Name = "Bem vindo!",
    Content = "Sua interface foi carregada com sucesso!",
    Time = 3
})

-- Componentes na aba principal --

-- Toggle (Interruptor)
Tab:AddToggle({
    Name = "Modo Hardcore",
    Default = false,
    Flag = "modoHardcore",
    Save = true,
    Callback = function(state)
        print("Modo Hardcore está:", state)
    end
})

-- Botão com notificação
Tab:AddButton({
    Name = "Clique aqui!",
    Callback = function()
        ServerUi:MakeNotification({
            Name = "Sucesso!",
            Content = "Você clicou no botão.",
            Time = 3
        })
    end
})

-- Slider (Controle deslizante)
Tab:AddSlider({
    Name = "Volume",
    Min = 0,
    Max = 100,
    Default = 50,
    Increment = 5,
    ValueName = "%",
    Flag = "volumeUser",
    Save = true,
    Callback = function(val)
        print("Volume:", val)
    end
})

-- Dropdown (Seleção de opções)
Tab:AddDropdown({
    Name = "Escolha sua cor favorita",
    Options = {"Vermelho", "Verde", "Azul", "Amarelo"},
    Default = "Vermelho",
    Flag = "corFavorita",
    Save = true,
    Callback = function(opt)
        print("Cor escolhida:", opt)
    end
})

-- Keybind (Atalho de tecla)
Tab:AddBind({
    Name = "Atalho para Ação",
    Default = Enum.KeyCode.G,
    Hold = false,
    Callback = function()
        print("Atalho ativado!")
    end
})

-- Campo de texto
Tab:AddTextbox({
    Name = "Digite algo",
    Default = "",
    TextDisappear = true,
    Callback = function(txt)
        print("Você digitou:", txt)
    end
})

-- Texto informativo
Tab:AddLabel("Este é um rótulo fixo de informação")

-- Parágrafo explicativo
Tab:AddParagraph("Aviso", "Este é um parágrafo com mais texto.\nVocê pode mostrar explicações, regras ou tutoriais aqui.")

-- Componentes na aba especial --

-- Log de sistema
Tab2:AddLog("Log do sistema iniciado com sucesso.")

-- Finalização e salvamento
ServerUi:Init()
Dica de Implementação: Este exemplo mostra como organizar uma interface completa com múltiplas abas, controle de acesso por cargos e todos os principais componentes interativos.

Dica Final

Organize seus elementos em abas, use permissões para esconder painéis sensíveis e explore as cores para uma UI intuitiva e bonita.
